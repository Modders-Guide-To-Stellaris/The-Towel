Hello Reader,

You might be wondering why the Towel repository has 3 folders. This is quite simple;
The Towel is built as a mod for the HGTS Stellaris Server, however it's separated in components to benefit everyone.
Additions are non-vanilla custom changes we've built, Balance is patches (as Vanilla as possible)

And the Towel of course, is the features the HGTS Council put into the mod plus some HGTS custom machinery be that as it may.

---

Each version of the Towel is hand-compiled, this meaning the user/creator is manually porting files from both mods into the towel.
Any mismatch errors are also custom edited in, so you might get the correctly build by just throwing files together... 
Unless of course the Towel has some custom framework running on it (or the aforementioned fixes/compat between mods).

That's all, any questions about the mod or this structure should be directed at the HGTS server
or the relevant creators. That being Curator Foxy and Sgt. Soncio.
